# Player-Data-Clustering
<br>
This project focuses on clustering player performance data using unsupervised learning techniques and applying classification to analyze grouped patterns.
<br>
Dataset Description
<br>
The dataset contains numerical player performance attributes such as:
<br>
Physical metrics
<br>
Performance statistics
<br>
Other structured numeric features
<br>
Only numerical features were used for clustering analysis.
<br>
Data Preprocessing
<br>
Removed missing values
<br>
Selected numeric features only
<br>
Standard data cleaning
